什么是PCA？

一.什么时候/为什么要用PCA?

当特征/因素之间又一定相关性，造成特征冗余，增加了计算量的时候；
高维数据可视化难度大，必须降低数据维度。

二.PCA算法

矩阵XXX均值+标准化
Xnew=X∗VX_{new}=X*VXnew​=X∗V,其中VVV是ev(XTX)或者ev=Cov(XT)ev(X^TX)或者ev=Cov(X^T)ev(XTX)或者ev=Cov(XT)排序后的前k列
k取值的标准：要求将(ew)\sqrt(ew)(​ew)后，前k个元素累加和cumsum>95%cumsum>95\%cumsum>95%

四.python 代码模板

import pandas as pd
import numpy as np
import matplotlib.pyplot as plt

#初始化读入数据
traindata_0=pd.read_csv(‘train_titanic.csv’,index_col=‘PassengerId’)
traindata_0.info()
traindata_0.head()

traindata=traindata_0.copy()

#处理数据，将标签量化
traindata.Sex=traindata.Sex.replace({‘female’:0,‘male’:1})
traindata.Embarked=traindata.Embarked.replace({‘C’:0,‘Q’:1,‘S’:2})

traindata.Age.fillna(traindata.Age.mean(),inplace=True)
traindata.Embarked.fillna(method=‘ffill’,inplace=True)

#把这几列数据去掉，因为结果没有直接的科学联系
traindata.drop(columns=[‘Survived’,‘Name’,‘Ticket’,‘Cabin’],inplace=True)
traindata.info()
print(traindata.head())

traindata.to_csv(‘data_pca_new.csv’)

#pca 开始处理
data=pd.read_csv(‘data_pca_new.csv’,index_col=‘PassengerId’)
data.describe()#查看基本信息（数量、均值、标准差…）
#标准化（得到的结果均值大约都是0，标准差大约都是1）
def norm_(x):
#求均值
xmean=np.mean(x,0)
#求标准差
std=np.std(x,0)
return (x-xmean)/std
data_=norm_(data)
data_.describe()

#特征值和特征向量
#ew,ev=np.linalg.eig(data_.T.dot(data_))
ew,ev=np.linalg.eig(np.cov(data_.T))
ew_order=np.argsort(ew)[::-1]
ew_sort=ew[ew_order]
ev_sort=ev[:,ew_order]
print(ew_sort)
print(ev_sort)
pd.DataFrame(ew_sort).plot(kind=‘bar’)#柱状图绘制

V=ev_sort[:,:2]
X_new=data_.dot(V)
get_ipython().run_line_magic(‘matplotlib’,‘notebook’)
sc=plt.scatter(X_new.iloc[:,0],X_new.iloc[:,1],s=5,c=traindata_0.Survived,cmap=plt.cm.coolwarm)
plt.xlabel(‘pc 0’)
plt.ylabel(‘pc 1’)
plt.colorbar(sc)

print(V)
print(data_.columns)
