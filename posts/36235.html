什么是K-means？

一.什么时候/为什么要用K-means？

  把n个点（可以是样本的一次观察或一个实例）划分到k个聚类中，使得每个点都属于离他最近的均值（此即聚类中心）对应的聚类，以之作为聚类的标准。
  应用场景有以下几个方面：
  ⋅\cdot⋅文档分类器
  ⋅\cdot⋅物品传输优化
  ⋅\cdot⋅识别犯罪地点
  ⋅\cdot⋅客户分类
  ⋅\cdot⋅球队状态分析
  ⋅\cdot⋅保险欺诈检测
  ⋅\cdot⋅乘车数据分析
  ⋅\cdot⋅网络分析犯罪分子
  ⋅\cdot⋅呼叫记录详细分析
  ⋅\cdot⋅IT警报的自动化聚类
k-means和k近邻没什么关系！！！

二.算法

  最常用的算法使用了迭代优化的技术。它被称为k-平均算法而广为使用，有时也被称为Lloyd算法（尤其在计算机科学领域）。已知初始的k个均值点m1(1),...,mk(1)m_1^{(1)},...,m_k^{(1)}m1(1)​,...,mk(1)​,算法的按照下面两个步骤交替进行:
  ⋅\cdot⋅分配(Assignment)
  将每个观测分配到聚类中，使得组内平方和（WCSS）达到最小。因为这一平方和就是平方后的欧氏距离，所以很直观地把观测分配到离它最近的均值点即可 [8] 。（数学上，这意味依照由这些均值点生成的Voronoi图来划分上述观测）。

Si(t)={xp:∥xp−mi(t)∥2⩽∥xp−mj(t)∥2∀j,1⩽j⩽k}S_i^{(t)}=\{x_p:\left \|x_p-m_i^{(t)} \right \|^2\leqslant\left \|x_p-m_j^{(t)} \right \|^2 \forall j,1\leqslant j\leqslant k\} Si(t)​={xp​:∥∥∥​xp​−mi(t)​∥∥∥​2⩽∥∥∥​xp​−mj(t)​∥∥∥​2∀j,1⩽j⩽k}

  其中每个xpx_pxp​都只被分配到一个确定的聚类StS^tSt中，尽管在理论上它可能被分配到2个或者更多的聚类。
  ⋅\cdot⋅更新(Update)
  对于上一步得到的每一个聚类，以聚类中观测值的图心，作为新的均值点。

三.python代码模板

import numpy as np
import matplotlib.pyplot as plt
import pandas as pd

data = pd.read_table(‘kmeans_data.txt’,header=None,names=[‘x’,‘y’])
x = data[‘x’]
y = data[‘y’]

plt.scatter(x,y)
data.head()

def distance(data,centers):
#data: 80x2, centers: 4x2
dist = np.zeros((data.shape[0],centers.shape[0]))
for i in range(len(data)):
for j in range(len(centers)):
dist[i,j] = np.sqrt(np.sum((data.iloc[i,:]-centers[j])**2))
return dist

def near_center(data,centers):
dist = distance(data,centers)
near_cen = np.argmin(dist,1)
return near_cen

def kmeans(data,k):
# 第一步:初始化中心点
centers = np.random.choice(np.arange(-5,5,0.1),(k,2))
print(centers)
for _ in range(10):
# 第二步: 点归属
near_cen = near_center(data,centers)
# 第三步：簇重心更新
for ci in range(k):
centers[ci] = data[near_cen==ci].mean()
return centers,near_cen

centers,near_cen = kmeans(data,4)
pic1=plt.scatter(x,y,c=near_cen)
plt.savefig(’./pic1’)
pic2=plt.scatter(centers[:,0],centers[:,1],marker=’.’,s=500,c=‘r’)
plt.savefig(’./pic2’)
